# Development guidelines

This is a vite typescript web-app using yarn as package manager. It uses prettier for formatting.

There is the import alias of `@/` which points to the `src` folder.

## Project structure:

- `src` - the source folder
  - `api` - the API clients library
  - `core` - the core business logic of the app
  - `features` - display components and associated logic
  - `third-party` - components that should be 3rd party components
